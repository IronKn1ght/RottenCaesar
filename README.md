# RottenCaesar
Caesar cipher decoder

This bash script decrypts plaintext ciphertexts in Caesar, taking into account all possible rotations of the ciphertext.

1. git clone https://github.com/IronKn1ght/RottenCaesar.git
2. cd RottenCaesar/
3. chmod +x ROT.SH
4. ./ROT.sh
